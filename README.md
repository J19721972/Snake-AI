# SnakeAI

**SnakeAI** is an AI agent that learns to play the classic Snake game using **Deep Q-Learning**. The project is built with **Python**, **PyGame**, and **PyTorch**, and includes real-time performance visualization.

---

## Features

- AI that learns to play Snake from scratch.
- Uses **reinforcement learning** with a neural network.
- Real-time display of the game using **PyGame**.
- Tracks scores and best record while training.

---
